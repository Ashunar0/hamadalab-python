# 球種ごとの予測精度評価の追加

## 概要
投球タイプ分類モデル（`pitch_classification_hanai.ipynb`）において、球種ごとの予測精度（Precision, Recall, F1-score）を詳細に評価・確認できるようにしました。

## 変更内容
- **Classification Reportの出力**: モデル評価セクションにて、各球種ごとの適合率・再現率・F1スコアを出力するようにしました。これにより、特定の球種（例: ストレート、スライダーなど）に対するモデルの得意・不得意が把握可能になります。
- **混同行列（Confusion Matrix）の可視化**: 実際の結果と予測結果の対比をヒートマップで表示し、どの球種がどの球種と誤分類されやすいかを視覚的に確認できるようにしました。

## 確認方法
1. `pitch_classification_hanai.ipynb` を実行する。
2. 「6. 予測の出力・評価」セクションまで進む。
3. `classification_report` の出力結果および混同行列のヒートマップを確認する。

## 補足
- 今回の変更により、全体の正答率（Accuracy）だけでなく、クラスごとの詳細な精度評価が可能になります。
