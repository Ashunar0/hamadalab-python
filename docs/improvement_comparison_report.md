# モデル改善比較レポート

## 概要
本レポートでは、特定の誤分類（特に `Others` と `FF`（4シーム・ファストボール）間）に対処するための改善を実施する前後の投球分類モデルの性能を比較します。

## 比較対象のノートブック
1.  **ベースライン（元データ）**: `asao_step0_1310-with_features_total_22.ipynb`
2.  **改善版**: `asao_improvement_others_ff.ipynb`

## 主な改善点

### 1. 「Others」対「FF」の誤分類
主な目標は、「Others」の投球が誤って「FF」と分類される数を減らすことでした。

*   **ベースライン（改善前）**:
    *   `Others` -> `FF` の誤分類数: **11件**（誤分類の主要因）
    *   根本原因: `Others` クラスには、`FF` と物理的に区別がつかない（球速 > 90mph、垂直変化量 > 0.8）データポイントが22件含まれていました。

*   **改善版**:
    *   `Others` -> `FF` の誤分類数: **2件**
    *   **削減率**: この特定の誤分類タイプにおいて **82% の削減**
    *   手法: トレーニング前に「FFに似たOthers」を `FF` として再ラベル付けする前処理ロジックを追加し、モデルを混乱させていた曖昧さを排除しました。

### 2. 全体的な正解率 (Accuracy)
*   **ベースラインの検証正解率**: 0.913
*   **改善版の検証正解率**: 0.913

*注: 「Others」クラスはデータセット全体のごく一部（0.02%）に過ぎないため、ここでの改善が全体の指標を劇的に変えることはありません。しかし、モデルの論理的な整合性は向上しています。*

## 誤分類の詳細分析（ワーストケース）

### ベースライン（誤分類の上位）
```
正解ラベル   予測ラベル   件数
    Others         FF     11  <-- 大きな課題
        SV         SL      4
        FC         FF      3
        KC         SL      3
```

### 改善版（誤分類の上位）
```
正解ラベル   予測ラベル   件数
        SI         FF      6
        SV         SL      5
        FA         FF      4
        ST         SL      3
        FC         FF      3
        FF         SI      3
    Others         FF      2  <-- 大幅に減少
```

## 結論
今回の修正により、モデル全体の性能を損なうことなく、特定の「Others」対「FF」の混同を緩和することに成功しました。残っている誤分類（SI対FF、SV対SLなど）は、今後の改善領域を示唆しており、これらのバイオメカニクス的に類似した球種を区別するための新しい特徴量が必要になる可能性があります。
